# مرجع CLI

يقدّم Foundry CLI ستة أوامر أساسية — واحد لكل أداة فرعية — بالإضافة إلى أوامر عليا لطَرق ونشر مساحات العمل.

## الاستخدام

```bash
foundry <command> [options]
```

## الأوامر العليا

| الأمر              | الوصف                                        |
|--------------------|----------------------------------------------|
| `foundry ignite`   | اطرق مساحة العمل من بيان `.grain`.           |
| `foundry deploy`   | اشحن الحزم إلى هدف نشر.                      |
| `foundry tongs`    | حلّ رسم التبعيات واعرضه بصريًا.              |
| `foundry quench`   | أدِر ذاكرة البناء التزايدية.                 |
| `foundry bellows`  | شغِّل مهام بالتوازي عبر حزم مساحة العمل.     |
| `foundry crucible` | ولِّد هياكل الاختبار وشغِّلها.               |
| `foundry slag`     | اكشف الشيفرة الميتة والتبعيات غير المستخدمة. |
| `foundry warden`   | شغِّل محرّك التدقيق.                         |

## foundry ignite

يقرأ بيان `.grain` ويُولِّد كل قطع مساحة العمل — بنية الدلائل، وملفات الإعداد، وقواعد Warden، وهياكل اختبار Crucible.

```bash title="Forge the workspace"
foundry ignite
```

| العلم           | الوصف                                        |
|-----------------|----------------------------------------------|
| `--dry-run`     | معاينة الخرج دون كتابة ملفات.                |
| `--incremental` | إعادة توليد الحزم ذات المداخل المتغيّرة فقط. |
| `--ci`          | خرج JSON مُهيكَل لدمج خطوط الأنابيب.         |
| `--verbose`     | طباعة سجلات تفصيلية لكل عملية.               |

```text title="Example output"
$ foundry ignite --verbose
[00:00.000] Loading manifest: project.grain
[00:00.012] Parsed workspace "my-app" (3 packages, 2 edges)
[00:00.018] Forging core (library)
[00:00.052] Forging api (service)
[00:00.091] Forging web (app)
[00:03.800] Done in 3.8s
```

## foundry tongs

يحلّ رسم التبعيات من البيان ويوفّر أدوات الاستعلام والعرض البصري.

```bash
foundry tongs <subcommand>
```

| الأمر الفرعي                      | الوصف                                  |
|-----------------------------------|----------------------------------------|
| `tongs graph`                     | اطبع رسم التبعيات.                     |
| `tongs graph --format mermaid`    | أخرج الرسم كمخطط Mermaid.              |
| `tongs verify`                    | افحص الدورات والمراجع المفقودة.        |
| `tongs query --depends-on <pkg>`  | اعرض الحزم التي تعتمد على حزمة مُعطاة. |
| `tongs query --build-order <pkg>` | اطبع ترتيب البناء لحزمة مُحدّدة.       |

## foundry quench

يدير ذاكرة البناء الموجّهة بالمحتوى.

```bash
foundry quench <subcommand>
```

| الأمر الفرعي                   | الوصف                                           |
|--------------------------------|-------------------------------------------------|
| `quench status`                | اعرض حالة الإصابة/الإخفاق للذاكرة لكل الحزم.    |
| `quench prune --older-than 7d` | أزِل مُدخلات الذاكرة الأقدم من المدة المُعطاة.  |
| `quench clear`                 | احذف الذاكرة بأكملها.                           |
| `quench stats`                 | اطبع حجم الذاكرة وعدد المُدخلات ومعدّل الإصابة. |

## foundry bellows

يُشغِّل المهام بالتوازي عبر حزم مساحة العمل، مع احترام رسم تبعيات Tongs.

```bash
foundry bellows <task> [options]
```

| العلم               | الوصف                               |
|---------------------|-------------------------------------|
| `--only <pkg>`      | شغِّل المهمة لحزمة واحدة وتبعياتها. |
| `--concurrency <n>` | تجاوز عدد العمّال المتوازيين.       |
| `--verbose`         | اطبع التوقيت والحالة لكل حزمة.      |

```text title="Example: parallel build"
$ foundry bellows build --verbose
  Wave 1: core (1.2s)
  Wave 2: auth, cli, docs (2.1s, parallel)
  Wave 3: api, web (1.8s, parallel)
  Total: 5.1s
```

## foundry crucible

يُولِّد هياكل اختبار من تواقيع الأنواع ويُشغِّل الاختبارات.

```bash
foundry crucible <subcommand>
```

| الأمر الفرعي                      | الوصف                                           |
|-----------------------------------|-------------------------------------------------|
| `crucible run`                    | شغِّل كل الاختبارات المولَّدة واليدوية.         |
| `crucible watch`                  | راقب التغييرات وأعِد تشغيل الاختبارات المتأثرة. |
| `crucible run --update-snapshots` | حدِّث كل ملفات اللقطات.                         |
| `crucible run --only <pkg>`       | شغِّل اختبارات حزمة واحدة.                      |

## foundry slag

يفحص مساحة العمل بحثًا عن الشيفرة الميتة والتصديرات غير المستخدمة والتبعيات اليتيمة.

```bash
foundry slag <subcommand>
```

| الأمر الفرعي  | الوصف                                                   |
|---------------|---------------------------------------------------------|
| `slag scan`   | افحص مساحة العمل وأبلِغ عن الشيفرة غير المستخدمة.       |
| `slag clean`  | أزِل الشيفرة الميتة المُكتشفة (يتطلب `--confirm`).      |
| `slag report` | ولِّد تقرير JSON لكل التصديرات والتبعيات غير المستخدمة. |

## foundry warden

يُشغِّل محرّك التدقيق على مساحة العمل.

```bash
foundry warden <subcommand>
```

| الأمر الفرعي                | الوصف                                        |
|-----------------------------|----------------------------------------------|
| `warden check`              | شغِّل كل القواعد وأبلِغ عن المخالفات.        |
| `warden fix`                | أصلح تلقائيًا كل المخالفات القابلة للإصلاح.  |
| `warden check --only <pkg>` | دقِّق حزمة واحدة.                            |
| `warden list`               | اعرض كل القواعد المُفعَّلة ومستويات خطورتها. |

## foundry deploy

يشحن حزم مساحة العمل إلى هدف نشر.

```bash
foundry deploy --target <target> [options]
```

| العلم              | الوصف                                        |
|--------------------|----------------------------------------------|
| `--target`         | هدف النشر: `arcline` أو `vial` أو `trellis`. |
| `--confirm`        | مطلوب للنشرات الإنتاجية.                     |
| `--rollback <ver>` | تراجع إلى إصدار مُحدّد.                      |
| `--verbose`        | اطبع سجلات نشر تفصيلية.                      |

## الوضع التفاعلي

يتضمّن Foundry وضع CLI تفاعليًا للاستكشاف والاكتشاف. أطلقه بـ:

```bash
foundry interactive
```

اختصارات لوحة المفاتيح في الوضع التفاعلي:

- <kbd>Ctrl</kbd> + <kbd>F</kbd> — البحث في الأوامر.
- <kbd>Ctrl</kbd> + <kbd>G</kbd> — عرض رسم التبعيات.
- <kbd>Ctrl</kbd> + <kbd>R</kbd> — إعادة تشغيل الأمر الأخير.
- <kbd>Tab</kbd> — الإكمال التلقائي لأسماء الأوامر والأعلام.
- <kbd>Ctrl</kbd> + <kbd>C</kbd> — الخروج من الوضع التفاعلي.

## الخيارات العامة

كل الأوامر تدعم هذه الأعلام:

| العلم       | الوصف                     |
|-------------|---------------------------|
| `--help`    | اعرض نص المساعدة لأي أمر. |
| `--version` | اطبع إصدار Foundry CLI.   |
