# التكاملات

يتحدّث Lantern بلغة HomeKit أوّلاً. وفيما عدا ذلك، يتحدّث بهدوء إلى الأشياء التي تعيش في البيت أصلاً.

## التكاملات المدمجة

- **HomeKit Home Hub** ـ الموزّع الأساسي، يُقرَن أثناء الإعداد
- **Threadbare** ـ للأتمتة المدفوعة بمستودعات يديرها أهل البيت
- **Spoke** ـ لتمرير الأحداث إلى نقاط نهاية داخلية
- **Canary** ـ للإشعارات التي يجب ألا تكون أعلى صوتاً مما يلزم

## التكاملات المخصَّصة

يمكن إضافة أي ملحق متوافق مع HomeKit إلى منطقة. لا يشترط Lantern شهادة اعتماد، بل يشترط أن يعرف الجهاز كيف يكون هادئاً.

## ضبط تكامل

```bash
lantern integration add canary --topic perimeter-evenings
```

كل تكامل لا يستقبل سوى الأحداث التي طلبها. لا شيء يُبثّ على الجميع.
