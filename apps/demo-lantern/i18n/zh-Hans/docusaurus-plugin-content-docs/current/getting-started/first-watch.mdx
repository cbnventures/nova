# 你的第一次看守

第一次看守，是围栏中最小的那一圈：一个分区，一台设备，一个安静的夜晚。

## 第一步：发现集线器

```bash
lantern hub discover
```

Lantern 会找到网络上的 HomeKit Home Hub，并不动声色地完成配对。

## 第二步：命名一个分区

```bash
lantern zone create perimeter --device front-door
```

只有一台设备的分区，依然是一个分区。Lantern 待它的方式，和待整间屋子一模一样。

## 第三步：为它布防

```bash
lantern arm perimeter
```

前门已经被看守。Lantern 不会出声，直到有事情发生——即便发生了，也只是轻声相告。

## 刚才发生了什么

你发现了集线器，命名了一个分区，并为它布防。门廊现在已属于这场看守。
