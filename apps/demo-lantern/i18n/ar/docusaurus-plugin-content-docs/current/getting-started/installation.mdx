# التثبيت

يُثبَّت Lantern من Loom Registry أو كحاوية Vial. اختر المسار الذي يثق به موزّعك أصلاً.

## المتطلّبات السابقة

- HomeKit Home Hub على الشبكة
- Spark 3.0 أو ما هو أحدث (لتثبيت سطر الأوامر)
- عشر دقائق هادئة، فأغلب التثبيتات تنتهي قبل الإبريق.

## التثبيت عبر Spark

```bash title="Install Lantern"
spark install lantern
```

تحقّق من التثبيت:

```bash
lantern --version
```

## التثبيت عبر Vial

```bash title="Pull the Lantern image"
vial pull lantern/cli:latest
```

شغِّله داخل طبقة الحاويات في شبكة المنزل. يبقى Lantern داخل غرفته الخاصة.
