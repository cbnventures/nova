# وضع السفر

يشدّ وضع السفر القواعد. تصبح المناطق الهادئة مناطق مراقبة. تنغلق نوافذ النوم. يتعلّم ضوء الشرفة أن يكون أكثر ارتياباً لأيام قليلة.

## الدخول إلى وضع السفر

```bash
lantern mode travel --until 2026-05-01
```

تاريخ الانتهاء إلزامي. لن يبقى Lantern متيقّظاً أطول مما تطلب.

## ما الذي يتغيّر

- **المناطق الهادئة** تكون مفعَّلة باستمرار
- **نطاقات الثقة** تضيق ـ الحركات الصغيرة تُحتسَب
- **التسليمات** تتطلّب عبارة مرور صريحة
- **الإشعارات** تُوجَّه إلى موزّع واحد موثوق به

## العودة إلى المنزل

ينتهي وضع السفر في التاريخ الذي حدّدته، أو أبكر إذا فعّلت `home` على جهاز معروف. يخفّ Lantern تلقائياً.

## الخطوات التالية

- [المناطق](/docs/zones/overview/) ـ كيف تتحوّل المناطق الهادئة إلى مناطق مراقبة أثناء السفر.
- [التسليمات](/docs/zones/handoffs/) ـ تشتدّ قواعد التسليم أثناء السفر.
- [التكاملات](/docs/zones/integrations/) ـ توجيه التنبيهات الموثوقة وأنت في الخارج.
