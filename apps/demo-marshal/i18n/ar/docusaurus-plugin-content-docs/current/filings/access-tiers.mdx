# فئات الوصول

يُصنِّف Marshal كلّ طلب وارد إلى فئة. الفئات غير قابلة للتفاوض — تُختَم عند الاستلام وتُوجَّه وفقًا لذلك.

## الفئات الثلاث

- **راعٍ** — الطابور قصير، والكاتب يردّ أوّلًا
- **مساهم** — الطابور هو الترتيب التشغيلي، بحسب تاريخ الإيداع
- **عامّ** — الطابور صبور، لكنّه ليس صامتًا أبدًا

## كيف يُطبَّق الختم

```bash
marshal classify --by github-sponsor,github-org-member
```

يقرأ Marshal حالة الطالب، ويُطبِّق ختم الفئة، ويُوجِّهه إلى المجلّد المناسب. الختم نفسه يُودَع في اللائحة.

## نسخ الفئة

```bash
marshal override request-id 4821 --tier contributor --reason "former maintainer"
```

كلّ نسخ هو سطر في دفتر القيد. لا ينسى Marshal من طلب الاستثناء، ولا لماذا.
