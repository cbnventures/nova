# 草稿示例

这是长文的草稿笔记。大致是说，在黑夜的街道上，一扇暖光的窗户所做的氛围工作，要远远多过任何门廊灯或泛光灯——即使它在物理上更小、亮度更低。

窗户能做这件事，是因为它暗示了有人居住——里面正发生着什么，并非只有你一人醒着。室外灯只能说明这栋房子还在。室内灯说明这栋房子在被使用。

我们想把这个想法接回到这一点：一份持续发布的变更日志，本身也是一扇被点亮的窗户——一个项目仍在里头活着的证据。
