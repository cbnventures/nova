# مسار التدقيق

مسار التدقيق هو اللائحة، مقروءة من أوّلها إلى آخرها. كلّ ختم، وكلّ إيداع، وكلّ نسخ — بترتيب وقوعها، مع توقيع الكاتب المُرفَق.

## قراءة المسار

```bash
marshal audit --since 90d --format ledger
```

المُخرَج عمود زمني وحيد. لا ملخّصات، ولا تجميعات. المسار مُعَدّ ليُقرأ، لا ليُتصفَّح.

## ما يلتقطه المسار

| الحقل       | الوصف                                           |
|-------------|-------------------------------------------------|
| `timestamp` | متى تصرّف Marshal، بدقّة الثانية.               |
| `clerk`     | أيّ نسخة من Marshal تولّت البند.                |
| `action`    | FILED، STAMPED، RECEIVED، ARCHIVED، OVERRIDDEN. |
| `policy`    | السياسة التي خوّلت الإجراء.                     |
| `subject`   | المستودع والمسار الذي طُبِّق عليه الإجراء.      |
| `seal`      | بصمة موقَّعة تربط هذا البند بالبند الذي يسبقه.  |

## التصدير للمراجعة

```bash
marshal audit --export quarterly-2026-q1.pdf
```

يُولِّد Marshal ملفّ PDF قابل للطباعة بختم الإدارة في أعلى كلّ صفحة. مناسب للإيداع.
