# 未列出样例

面向新入职 intake 事务员的早期指引文档。

我们最常见的两类失误：一是过度依赖文件类别下拉框——事务员遇到不确定时默认选择"杂项"，结果让下游检索形同虚设；二是欠用交叉引用字段——事务员往往跳过它，因为它是可选项，可正是这一字段，让我们在日后纠纷浮现时能把相关 filing 重新串联起来。

两者的解法是同一个：拿不准时，在提交前向当值的资深事务员问一句。打扰别人比重新立卷要快。五分钟打扰原则是真章，不是建议。
