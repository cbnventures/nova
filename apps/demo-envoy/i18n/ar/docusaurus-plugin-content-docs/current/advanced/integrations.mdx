# التكاملات

يأتي Envoy مع مترجمات مُدمَجة لمصادر البنية التحتية الشائعة. يفهم كلّ مترجم صيغة حمولة المصدر، وأنواع الأحداث، وآلية المصادقة. المصادر المخصّصة مدعومة عبر المترجم العامّ.

<TabItem value="threadbare" label="Threadbare" default>

## Threadbare

Threadbare منصّة استضافة لشيفرة المصدر تُطلق خطّافات ويب عند أحداث المستودع. يتعامل مترجم Threadbare في Envoy مع أحداث الدفع، وطلبات السحب، والإصدارات، وتحديثات القضايا.

### الأحداث المدعومة

| الحدث          | حقول الحمولة                                    | الموضوع الافتراضي |
|----------------|-------------------------------------------------|-------------------|
| `push`         | `repo`, `branch`, `author`, `commits`, `count`  | `ci-builds`       |
| `pull_request` | `repo`, `action`, `title`, `author`, `number`   | `ci-builds`       |
| `release`      | `repo`, `tag`, `name`, `author`, `html_url`     | `deploy-alerts`   |
| `issues`       | `repo`, `action`, `title`, `number`, `assignee` | `project-updates` |

### مثال

```text title="relay.grain — Threadbare integration"
relay "threadbare-pushes" {
  source = "threadbare"
  cipher = "hmac-sha256"

  cipher_config {
    header = "X-Threadbare-Signature-256"
    secret = "${THREADBARE_SECRET}"
  }

  transform {
    title = "[{{ repo }}] {{ count }} commit(s) to {{ branch }}"
    body  = "{{ author }}: {{ commit_summary }}"
  }

  destination = "canary://ci-builds"
}
```

**الحمولة الواردة:**

```json title="Threadbare push event"
{
  "ref": "refs/heads/main",
  "repository": { "full_name": "acme/api" },
  "pusher": { "name": "anders" },
  "commits": [
    { "id": "c3d2a1f", "message": "Add Courier health endpoint" }
  ]
}
```

**المخرَج المُحوَّل:**

```json title="Canary notification"
{
  "title": "[acme/api] 1 commit(s) to main",
  "body": "anders: c3d2a1f — Add Courier health endpoint",
  "priority": 3,
  "tags": ["package"]
}
```

## Glassboard

Glassboard منصّة رصد ولوحات مراقبة تُطلق خطّافات ويب عند تغيّر حالات التنبيه. يتعامل مترجم Glassboard في Envoy مع إشعارات التنبيه وأحداث التعافي.

### الأحداث المدعومة

| الحدث      | حقول الحمولة                                            | الموضوع الافتراضي |
|------------|---------------------------------------------------------|-------------------|
| `alerting` | `alertname`, `severity`, `instance`, `message`, `value` | `infra-alerts`    |
| `resolved` | `alertname`, `severity`, `instance`, `resolved_at`      | `infra-alerts`    |

### ربط درجات الخطورة

| درجة Glassboard | أولوية Canary | توصية التوجيه   |
|-----------------|---------------|-----------------|
| `critical`      | 5             | `oncall-urgent` |
| `high`          | 4             | `infra-alerts`  |
| `warning`       | 3             | `infra-alerts`  |
| `info`          | 2             | `infra-log`     |

### مثال

```text title="relay.grain — Glassboard integration"
relay "glassboard-critical" {
  source = "glassboard"
  cipher = "hmac-sha256"

  filter { severity = "critical" }

  transform {
    title    = "[critical] {{ alertname }}"
    body     = "{{ instance }} — {{ message }}"
    priority = 5
    tags     = ["rotating_light"]
  }

  destination = "canary://oncall-urgent"
}
```

## Canary

Canary خدمة رصد توفّر تُطلق خطّافات ويب حين تتغيّر حالات النقاط المُراقَبة. يتعامل مترجم Canary في Envoy مع أحداث التعطّل والتعافي.

### الأحداث المدعومة

| الحدث            | حقول الحمولة                                      | الموضوع الافتراضي |
|------------------|---------------------------------------------------|-------------------|
| `down`           | `monitor_name`, `url`, `status_code`, `timestamp` | `monitoring`      |
| `up`             | `monitor_name`, `url`, `downtime_seconds`         | `monitoring`      |
| `heartbeat_fail` | `monitor_name`, `expected_interval`, `last_seen`  | `monitoring`      |

### مثال

```text title="relay.grain — Canary integration"
relay "canary-down" {
  source = "canary"
  cipher = "bearer"

  filter { event = "down" }

  transform {
    title    = "[DOWN] {{ monitor_name }}"
    body     = "{{ url }} returned {{ status_code }} at {{ timestamp }}"
    priority = 5
    tags     = ["red_circle"]
  }

  destination = "canary://oncall-urgent"
}
```

## المصادر المخصّصة

أيّ نظام يستطيع إرسال HTTP POST بحمولة JSON أو XML أو form-data يستطيع التكامل مع Envoy عبر المترجم العامّ.

### مثال

```text title="relay.grain — custom source"
relay "custom-deploy-hook" {
  source = "generic"
  cipher = "bearer"

  cipher_config {
    token = "${DEPLOY_WEBHOOK_TOKEN}"
  }

  transform {
    title = "Deploy: {{ app }} → {{ environment }}"
    body  = "Version {{ version }} deployed by {{ user }}"
    priority = 3
  }

  destination = "canary://deploy-alerts"
}
```

يستخرج المترجم العامّ الحقول من كائن JSON على المستوى العلوي. الحقول المتداخلة تستخدم تدوين النقطة: `{{ server.hostname }}`.

:::note
تتطلّب المصادر المخصّصة تعيينًا صريحًا للحقول داخل كتلة التحويل. وبخلاف التكاملات المُدمَجة، لا يوجد تطبيع تلقائي للحمولة — أنت تُعيّن الحقول التي تحتاجها بالضبط.
:::
