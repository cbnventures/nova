# نموذج مسودة

ملاحظات عمل حول إعادة تصميم فاصل المسبارات. الإعدادات الافتراضية الحالية هي 60 ثانية لمسبارات HTTP و30 ثانية لمسبارات TCP.

في الشهر الماضي واجهنا حادثة جار ضاجّ، حيث شغّلت خدمتان على نفس العقدة فحوصاتهما في الوقت ذاته، فاستدعت العقدة المضيفة الفريق بسبب حِمل اصطناعي صادر منّا بالكامل.

الخطة هي إضافة نافذة تشويش لكلّ عقدة بمقدار يزيد أو ينقص 8 ثوانٍ، بحيث تنحرف المسبارات طبيعياً عن بعضها، مع اشتقاق بذرة التشويش من اسم مضيف العقدة لتبقى ثابتة عبر إعادات التشغيل. ولا نريد إطلاق هذا قبل محاكاته مقابل نوافذ تقييم التنبيهات، فالسلبيات الخاطئة ستكون أسوأ بكثير ممّا لدينا الآن.
