# مراقبتك الأولى

المراقبة الأولى هي أصغر محيط ممكن: منطقة واحدة، وجهاز واحد، وأمسية هادئة واحدة.

## الخطوة 1: اكتشف الموزّع

```bash
lantern hub discover
```

يحدّد Lantern موقع HomeKit Home Hub على الشبكة ويقترن به من دون عناء.

## الخطوة 2: سمِّ منطقة

```bash
lantern zone create perimeter --device front-door
```

المنطقة بجهاز واحد لا تزال منطقة. يتعامل Lantern معها بالطريقة نفسها التي يتعامل بها مع البيت كله.

## الخطوة 3: فعِّلها

```bash
lantern arm perimeter
```

باب المدخل صار مراقَباً. لا يقول Lantern شيئاً حتى يحدث شيء، وعندها يقوله بلطف.

## ما الذي حدث للتوّ

اكتشفت موزّعاً، وسمّيت منطقة، وفعّلتها. الشرفة أصبحت الآن جزءاً من المراقبة.
