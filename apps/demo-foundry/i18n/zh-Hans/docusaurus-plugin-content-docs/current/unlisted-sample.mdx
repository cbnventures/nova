# 未列出示例

标准目录件出现缺货时的升级处理顺序。

对于 304 等级的棒料与条料，第一通电话永远是 Hartline Mill——区域供应商，四天交期，柜台上接电话的是 John 或 Erin。如果 Hartline 七天内无法供货，第二通电话打给 Northern Continental——价格更高，但他们持有我们医疗器械产线所需的认证文件。

最后的选择是采购团队每月汇总里的现货市场清单，但这条路烧过我们两次，最后总是比直接等正常供应商更慢。电话号码记在文件夹里，不放在这份文档中。
