# 欢迎使用 Lantern

Lantern 是一套面向 HomeKit 集线器的全屋监护工具——它更偏爱温暖的光，而非闪烁的红。

整间屋子，被温柔地照看着。从前门廊到后过道，每一处分区都被注视，却又始终不让人感觉被窥视。

## 你可以做什么

- 在任意分区中发现并布防设备
- 实时接收正在发生的动作事件
- 不出客厅便能浏览传感器历史
- 把布防 / 撤防的工作，凌晨两点交给你信任的人

没有任何配置是一盏门廊灯不该知道的。
