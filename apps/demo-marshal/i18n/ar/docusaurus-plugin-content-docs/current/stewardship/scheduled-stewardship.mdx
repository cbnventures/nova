# الرعاية المجدولة

يُشغِّل Marshal الأعمال المجدولة على cron. التنظيف، التدقيق، التجديد، مسح التسميات — أيّ شيء ينبغي أن يحدث في رأس الساعة دون إيقاظ أحد.

## ما الذي يمكن جدولته

- **التنظيف** — إغلاق وختم المسارات التي تستوفي شرط السياسة
- **التدقيق** — التحقّق من أنّ كلّ تذكرة مفتوحة مُصنَّفة
- **التجديد** — دفع اللائحة إلى الأمام نحو الربع التالي
- **مسح التسميات** — تطبيق أو إزالة التسميات بالجملة

## إيداع عمل مجدول

```yaml title="schedules/quarterly-rollover.yml"
schedule: "0 0 1 1,4,7,10 *"
action: rollover
docket: "FY-{quarter}"
notice: "Rolled forward at the top of the quarter."
```

## مراقبة cron

```bash
marshal cron list
```

يُدرِج Marshal كلّ عمل مجدول، وموعد تشغيله التالي، والسياسة التي أودَعَته. لا يُشغَّل شيء غير مختوم.

## الخطوات التالية

- [السياسات](/docs/filings/policies/) — جدولة سياسة للإيداع التلقائي.
- [اللوائح](/docs/dockets/overview/) — حيث تترك كلّ تشغيلة مجدولة سطرها.
- [مسار التدقيق](/docs/dockets/audit-trail/) — التدقيقات الفصليّة للأعمال المجدولة.
