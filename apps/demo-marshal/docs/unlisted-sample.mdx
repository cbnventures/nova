# Unlisted Sample

Early-orientation document for new intake clerks.

The two failure modes we see most often are over-reliance on the document-class dropdown — clerks default to "miscellaneous" when uncertain, which makes downstream search useless — and under-reliance on the cross-reference field, which clerks skip because it is optional but which is precisely the field that lets us reconnect related filings later when a dispute surfaces.

The fix for both is the same: when in doubt, ask the senior clerk on shift before submitting. It is faster to interrupt someone than to refile a record. The five-minute interruption rule is real and not a suggestion.
