# Draft Sample

Scratch notes for the longer essay. Something about how a single warm window in a dark street does more atmospheric work than any porchlight or floodlight, even though it is physically smaller and lower in lumens.

The window is doing the work because it implies habitation — something is happening inside, you are not the only one awake. Outdoor lights only signal that the house exists. Indoor lights signal that the house is in use.

We want to tie this back to the idea that a shipping changelog is a kind of lit window — proof the project is alive in there.
