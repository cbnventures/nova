# 欢迎来到 Marshal

Marshal 是仓储事务部——为偏爱文书甚于喧嚣的项目提供自动化的 stewardship。

立卷一式三份。本着诚意加盖封印。正午前完成钤章。

## 你能做什么

- 为你 stewardship 下的任何仓库提交 policy 工作流
- 按计划归档静默线程，并留下书面凭据
- 按访问层级把守支持与分流——不必亲自动笔
- 把每一项操作记录到公开 ledger——任何事都不会悄然合上

这里没有 1962 年的事务员认不出来的东西。
