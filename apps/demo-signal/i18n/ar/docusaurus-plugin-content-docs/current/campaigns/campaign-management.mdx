# إدارة الحملات

يلتقط Aperture لقطات للأداء عند نقاط زمنية محددة من الحملة. قارن اللقطات عبر النوافذ الزمنية. شغّل صيغ روابط A/B. اطّلع بدقة على ما تغيّر ومتى.

## إنشاء الحملات

اجمع روابط Beacon في حملات لتتبّع منظَّم:

```bash title="إنشاء حملة"
signal campaign create \
  --name "product-launch" \
  --description "Q1 product launch across email, social, and paid channels" \
  --start "2025-02-01" \
  --end "2025-03-31"
```

```json title="استجابة الحملة"
{
  "id": "cmp_01J9L2M8N4P6Q7R3S5T0U1V9",
  "name": "product-launch",
  "description": "Q1 product launch across email, social, and paid channels",
  "window": {
    "start": "2025-02-01T00:00:00Z",
    "end": "2025-03-31T23:59:59Z"
  },
  "beacons": 0,
  "clicks": 0,
  "status": "active"
}
```

كل رابط Beacon يُنشأ بـ `--campaign "product-launch"` يُربط تلقائياً بهذه الحملة.

## لقطات Aperture

يلتقط Aperture حالة الحملة في لحظة محددة. اللقطات غير قابلة للتعديل، تسجل المقاييس كما كانت، لا كما هي الآن.

```bash title="التقاط لقطة"
signal aperture snapshot --campaign "product-launch" --label "week-2"
```

```json title="لقطة Aperture"
{
  "id": "snap_01J9M3N5P7Q8R4S6T2U0V3W1",
  "campaign": "product-launch",
  "label": "week-2",
  "capturedAt": "2025-02-14T18:00:00Z",
  "metrics": {
    "totalBeacons": 24,
    "totalClicks": 12847,
    "uniqueVisitors": 9210,
    "conversionRate": 0.034,
    "topChannel": "email",
    "topVariant": "hero-cta"
  }
}
```

## مقارنة اللقطات

قارن لقطتين لمعرفة ما تغيّر بين النافذتين الزمنيتين:

```bash title="مقارنة اللقطات"
signal aperture compare --from "week-1" --to "week-2" --campaign "product-launch"
```

```json title="مخرَج المقارنة"
{
  "campaign": "product-launch",
  "from": "week-1",
  "to": "week-2",
  "deltas": {
    "totalClicks": { "from": 5420, "to": 12847, "change": "+136.8%" },
    "uniqueVisitors": { "from": 3890, "to": 9210, "change": "+136.8%" },
    "conversionRate": { "from": 0.021, "to": 0.034, "change": "+61.9%" },
    "topChannel": { "from": "social", "to": "email", "changed": true }
  }
}
```

## صيغ روابط A/B

أنشئ صيغ متعددة لروابط Beacon لنفس الوجهة لاختبار الرسائل:

<Tabs>
<TabItem value="single" label="حملة واحدة" default>

```bash title="إنشاء صيغ A/B"
signal beacon create \
  --url "https://threadbare.example/pricing" \
  --campaign "product-launch" \
  --channel "email" \
  --variant "pricing-direct"

signal beacon create \
  --url "https://threadbare.example/pricing" \
  --campaign "product-launch" \
  --channel "email" \
  --variant "pricing-soft-sell"
```

استعلم عن أداء الصيغ ضمن الحملة:

```bash title="مقارنة الصيغ"
signal aperture variants --campaign "product-launch" --channel "email"
```

```json title="مقارنة الصيغ"
{
  "channel": "email",
  "variants": [
    { "variant": "pricing-direct", "clicks": 1842, "conversions": 73, "rate": 0.040 },
    { "variant": "pricing-soft-sell", "clicks": 1956, "conversions": 41, "rate": 0.021 }
  ]
}
```

</TabItem>
<TabItem value="cross" label="عبر الحملات">

قارن الصيغة نفسها عبر حملات متعددة:

```bash title="استعلام صيغ عبر الحملات"
signal aperture variants \
  --variant "hero-cta" \
  --campaigns "product-launch,summer-sale,year-end"
```

```json title="مخرَج عبر الحملات"
{
  "variant": "hero-cta",
  "campaigns": [
    { "campaign": "product-launch", "clicks": 4210, "conversions": 143, "rate": 0.034 },
    { "campaign": "summer-sale", "clicks": 6890, "conversions": 289, "rate": 0.042 },
    { "campaign": "year-end", "clicks": 3150, "conversions": 98, "rate": 0.031 }
  ]
}
```

</TabItem>

## التحليل بنوافذ زمنية

قَيّد الاستعلامات بنطاقات زمنية محددة:

```bash title="استعلام نافذة زمنية"
signal aperture metrics \
  --campaign "product-launch" \
  --from "2025-02-10" \
  --to "2025-02-14" \
  --granularity "day"
```

| التاريخ    | النقرات | الزوار الفريدون | التحويلات | المعدل |
|------------|---------|-----------------|-----------|--------|
| 2025-02-10 | 1,240   | 890             | 28        | 3.1%   |
| 2025-02-11 | 1,680   | 1,210           | 42        | 3.5%   |
| 2025-02-12 | 2,150   | 1,540           | 61        | 4.0%   |
| 2025-02-13 | 1,890   | 1,350           | 48        | 3.6%   |
| 2025-02-14 | 1,520   | 1,090           | 34        | 3.1%   |
