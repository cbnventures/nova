# 集成

Lantern 首先用 HomeKit 交流。除此之外，它会轻声地与屋子里已经住下的其他东西对话。

## 内置集成

- **HomeKit Home Hub** — 主集线器，在安装时完成配对
- **Threadbare** — 用于由家庭维护的、以仓库为驱动的自动化
- **Spoke** — 将事件中继至内部端点
- **Canary** — 提供通知，且永远不会喧哗超出必要

## 自定义集成

任何兼容 HomeKit 的配件都可以加入分区。Lantern 不要求认证——它只要求设备知道如何保持安静。

## 配置一项集成

```bash
lantern integration add canary --topic perimeter-evenings
```

每项集成只接收它所要求的事件。没有任何广播。
