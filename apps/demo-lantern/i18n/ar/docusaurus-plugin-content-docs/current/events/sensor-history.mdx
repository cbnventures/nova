# سجل المستشعرات

سجل المستشعرات هو السجلّ الهادئ لكل حركة واتصال وتغيير منطقة شاهده Lantern. اقرأه عند الظهيرة. اقرأه بعد عطلة نهاية أسبوع طويلة. يُبقي الفانوس الصفحة مفتوحة.

## ما يلتقطه السجل

- **أقواس الحركة** ـ الاتجاه، والثقة، ووقت اليوم
- **مدّة الاتصال** ـ كم بقي باب الخلف محجوزاً
- **إيقاع التفعيل والإلغاء** ـ من فعَّل ومتى، ومن ألغى ومتى
- **الفترات الهادئة** ـ الفراغات بين الأحداث

## قراءة السجل

```bash
lantern history --zone perimeter --since 7d
```

الناتج عمود واحد من الأوقات والأفعال اللطيفة. لا شيء يُختزَل ويُحذَف.
