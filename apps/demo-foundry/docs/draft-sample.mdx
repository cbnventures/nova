# Draft Sample

Comparison notes for the next batch of fittings. 304 stainless gives us the cost we want and handles indoor potable water fine, but the customer's bid spec calls for 316 because the lines run through a coastal facility with chloride exposure.

The price delta is roughly thirty percent on the rod stock and another fifteen on the bar. Rough math says we eat a four-thousand-dollar hit per assembly if we cross-grade, which is survivable on this run but corrosive on margin if it becomes the default.

We need to ask procurement whether the marine spec is contractual or aspirational before we cut anything.
