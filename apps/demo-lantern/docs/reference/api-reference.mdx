# API Reference

The Lantern Spoke API is small on purpose. Most households never need it.

## Base URL

```
https://lantern.local/api
```

## Endpoints

| Method | Path                      | Description                          |
|--------|---------------------------|--------------------------------------|
| `GET`  | `/api/zones`              | List every zone Lantern is watching. |
| `POST` | `/api/zones/:name/arm`    | Arm a zone by name.                  |
| `POST` | `/api/zones/:name/disarm` | Disarm a zone by name.               |
| `GET`  | `/api/events`             | Replay the event stream.             |
| `GET`  | `/api/history`            | Read the sensor history ledger.      |
| `GET`  | `/api/mode`               | Get the current household mode.      |
| `POST` | `/api/mode`               | Set travel, bedtime, or normal mode. |

## Example

```bash
curl https://lantern.local/api/zones \
  -H "Authorization: Bearer $LANTERN_TOKEN"
```

The response lists every zone, its devices, and whether it is currently armed. Nothing more, nothing louder.
