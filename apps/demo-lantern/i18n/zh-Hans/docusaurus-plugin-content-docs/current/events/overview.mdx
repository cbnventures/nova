# 事件

每一次动作、接触、布防与撤防，都会经过 Lantern 的事件流。可回放、可过滤、可移交给集线器。没有遗漏，也没有喧哗。

## 事件类型

- **动作（Motion）** — 带方向，并可附置信带
- **接触（Contact）** — 门或窗的开启、关闭、保持
- **布防 / 撤防（Arm / disarm）** — 按分区、按用户、按时间表
- **传感器历史（Sensor history）** — 一本可以在正午翻阅的安静账册
