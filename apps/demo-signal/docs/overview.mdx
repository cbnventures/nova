# Signal

Signal is an attribution engine for the attention economy. It traces how intent travels — from first encounter to final action. Every link is a sensor. Every redirect is a data point. Signal tells you not just what was clicked, but why, and what almost happened instead.

> "The shortest distance between a click and an insight is a link that knows where it has been."

## What Signal Does

Most analytics tools count events after the fact. Signal instruments the path itself. Each link minted through Signal carries embedded attribution metadata that survives URL shortening, messaging apps, private browsing, and cross-device handoffs. When a conversion happens, Signal reconstructs the full journey — every touchpoint, every hesitation, every channel that contributed.

| Tool          | Purpose                                                                           |
|---------------|-----------------------------------------------------------------------------------|
| **Beacon**    | Link minting engine — branded short links with embedded attribution metadata.     |
| **Prism**     | Multi-touch attribution modeler — traces conversion paths across channels.        |
| **Pulse**     | Real-time click stream — live feed of interactions, geo, device, and referrer.    |
| **Aperture**  | Campaign snapshots — point-in-time performance captures with comparison overlays. |
| **Resonance** | Audience overlap detector — finds where audiences bleed into each other.          |
| **Flare**     | QR and deep link generator — physical-to-digital bridge with full attribution.    |

:::info
Signal operates at the link level. Other attribution platforms bolt tracking onto pages. Signal embeds it into the links themselves — so attribution data exists before the page even loads.
:::

## How It Works

A Beacon link is not just a redirect. It encodes the campaign, the channel, the variant, and a unique trace identifier. When the link is clicked, Pulse captures the interaction in real time. When the user eventually converts, Prism walks the full attribution path backward and distributes credit across every touchpoint.

```
Beacon mints link → User clicks → Pulse captures event
  → User clicks again from a different channel → Pulse captures
    → User converts → Prism reconstructs the full path
      → Aperture snapshots the campaign state at conversion time
```

## When to Use Signal

- **Campaign attribution** — trace which channels, links, and touchpoints drive conversions.
- **Audience intelligence** — detect overlap between campaigns and segments with Resonance.
- **Real-time monitoring** — watch click streams arrive live with Pulse.
- **Physical-to-digital** — bridge print, out-of-home, and events to digital attribution with Flare.
- **Dark traffic recovery** — recover attribution for clicks that strip referrer and UTM data.
