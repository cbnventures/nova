# Integrations

Lantern speaks HomeKit first. Beyond that, it speaks softly to the things that already live in the house.

## Built-in integrations

- **HomeKit Home Hub** — the primary hub, paired during setup
- **Threadbare** — for repository-driven automations the household maintains
- **Spoke** — to relay events to internal endpoints
- **Canary** — for notifications that should never be louder than necessary

## Custom integrations

Any HomeKit-compatible accessory can be added to a zone. Lantern does not require certification — it requires that the device knows how to be quiet.

## Configuring an integration

```bash
lantern integration add canary --topic perimeter-evenings
```

Each integration receives only the events it asked for. Nothing is broadcast.
