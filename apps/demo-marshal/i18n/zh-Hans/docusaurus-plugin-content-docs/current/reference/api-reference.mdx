# API 参考

Marshal 提供一个 Spoke API，用于提交 policy、通读 docket 和导出审计轨迹。Bearer 令牌认证、JSON 响应，没有意外。

## Base URL

```
https://marshal.your-org.example/api
```

## 端点

| Method   | Path                 | 说明                     |
|----------|----------------------|------------------------|
| `GET`    | `/api/policies`      | 列出所有已提交的 policy。       |
| `POST`   | `/api/policies`      | 提交一条新 policy。          |
| `DELETE` | `/api/policies/:id`  | 撤回一条 policy（以覆盖立卷）。    |
| `GET`    | `/api/dockets`       | 列出跨仓库的 docket。         |
| `GET`    | `/api/dockets/:repo` | 通读某一个仓库的 docket。       |
| `POST`   | `/api/audit/export`  | 生成一次审计轨迹导出。            |
| `GET`    | `/api/cron`          | 列出计划中的 stewardship 任务。 |

## 示例

```bash
curl https://marshal.your-org.example/api/dockets/org/api \
  -H "Authorization: Bearer $MARSHAL_TOKEN"
```

响应是该仓库上每一次立卷操作的时间顺序列表。官署封印出现在每一条记录里。
