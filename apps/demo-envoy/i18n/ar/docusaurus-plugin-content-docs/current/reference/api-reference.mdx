# مرجع الواجهة البرمجية

يكشف Envoy عن Spoke API لإدارة المُرحِّلات، ورصد الصحّة، وفحص التسليم، وإدارة المواضيع. كلّ النقاط تُعيد JSON. وتستخدم المصادقة آليات Cipher نفسها المُعدّة لنقاط المُرحِّلات.

## عنوان الأساس

```
http://localhost:8090/api
```

كلّ النقاط أدناه نسبية إلى هذا العنوان.

## نقاط المُرحِّل

### إرسال رسالة

```bash title="POST /api/relay/:topic"
curl -X POST http://localhost:8090/api/relay/infra-alerts \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer ${RELAY_TOKEN}" \
  -d '{
    "title": "Manual alert",
    "body": "Triggered via API.",
    "priority": 3
  }'
```

```json title="Response — 200 OK"
{
  "id": "msg_a1b2c3d4",
  "status": "delivered",
  "relay": "manual",
  "latency_ms": 2.8
}
```

### الحصول على حالة المُرحِّل

```bash title="GET /api/relay/:name/status"
curl http://localhost:8090/api/relay/threadbare-pushes/status
```

```json title="Response — 200 OK"
{
  "relay": "threadbare-pushes",
  "active": true,
  "messages_today": 47,
  "delivery_rate": 1.0,
  "last_delivery": "2025-08-22T14:30:07Z",
  "dead_lettered": 0
}
```

## نقاط المواضيع

### سرد المواضيع

```bash title="GET /api/topics"
curl http://localhost:8090/api/topics
```

```json title="Response — 200 OK"
{
  "topics": [
    { "name": "ci-builds", "relays": 2, "messages_today": 31 },
    { "name": "infra-alerts", "relays": 3, "messages_today": 12 },
    { "name": "oncall-urgent", "relays": 1, "messages_today": 4 },
    { "name": "monitoring", "relays": 2, "messages_today": 0 }
  ]
}
```

### إنشاء موضوع

```bash title="POST /api/topics"
curl -X POST http://localhost:8090/api/topics \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer ${ADMIN_TOKEN}" \
  -d '{ "name": "deploy-alerts", "retention": "30d" }'
```

```json title="Response — 201 Created"
{
  "topic": "deploy-alerts",
  "created": true,
  "retention": "30d"
}
```

### تحديث موضوع

```bash title="PATCH /api/topics/:name"
curl -X PATCH http://localhost:8090/api/topics/deploy-alerts \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer ${ADMIN_TOKEN}" \
  -d '{ "retention": "90d" }'
```

### حذف موضوع

```bash title="DELETE /api/topics/:name"
curl -X DELETE http://localhost:8090/api/topics/deploy-alerts \
  -H "Authorization: Bearer ${ADMIN_TOKEN}"
```

```json title="Response — 200 OK"
{
  "topic": "deploy-alerts",
  "deleted": true
}
```

### إعادة محاولة رسالة ميّتة

```bash title="POST /api/deliveries/:id/retry"
curl -X POST http://localhost:8090/api/deliveries/msg_a9b8c7d6/retry \
  -H "Authorization: Bearer ${ADMIN_TOKEN}"
```

{
  "id": "msg_a9b8c7d6",
  "status": "pending",
  "message": "Message re-queued for delivery."

## ملخّص النقاط

| الطريقة  | المسار                      | الوصف                               |
|----------|-----------------------------|-------------------------------------|
| `POST`   | `/api/relay/:topic`         | إرسال رسالة إلى موضوع.              |
| `GET`    | `/api/relay/:name/status`   | الحصول على حالة المُرحِّل ومقاييسه. |
| `GET`    | `/api/health`               | فحص الصحّة.                         |
| `GET`    | `/api/health/metrics`       | مقاييس تفصيلية للمنصّة.             |
| `GET`    | `/api/topics`               | سرد جميع المواضيع.                  |
| `POST`   | `/api/topics`               | إنشاء موضوع جديد.                   |
| `PATCH`  | `/api/topics/:name`         | تحديث إعداد الموضوع.                |
| `DELETE` | `/api/topics/:name`         | حذف موضوع.                          |
| `GET`    | `/api/deliveries`           | سرد التسليمات مع التصفية.           |
| `POST`   | `/api/deliveries/:id/retry` | إعادة محاولة رسالة ميّتة.           |
| `GET`    | `/api/deliveries/:id`       | فحص تسليم واحد.                     |
