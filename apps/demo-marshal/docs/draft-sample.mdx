# Draft Sample

Notes for the v3 intake form rework. The current form asks for jurisdiction, filing party, document class, and reference docket — but we have been getting back-and-forth with submitters who do not know what document class their record falls into, and the current dropdown has thirty-one options.

The path forward is probably either collapsing the class taxonomy down to seven top-level groupings with progressive disclosure, or replacing the dropdown with a free-text field reconciled at intake review.

Either way the back-and-forth is costing two business days per submission on average. We will pull last quarter's metrics before deciding.
