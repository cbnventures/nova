# Lantern

Warm amber and indigo tones paired with Fraunces display headings, Manrope body text, and IBM Plex Mono for code blocks. Rounded shapes with comfortable spacing and elevated cards with soft shadows.

## Why Use This Preset?

- Warm amber palette so home-monitoring and smart-home toolkit documentation feels inviting and domestic rather than clinical.
- Fraunces display headings paired with Manrope body text so the typography balances warmth with structured readability.
- Elevated cards with staggered reveals so content sections feel layered and polished across device dashboards and setup guides.
- Bridge navbar carries Lantern's amber-underline-glow signature — a subtle eaves-like accent that frames navigation without overpowering it.

## Summary

Lantern is designed for whole-house monitoring and home-automation documentation. The warm amber palette communicates familiarity while the elevated card depth adds visual hierarchy. Motion is set to normal speed with staggered reveals and hover effects enabled.

  // Default values (no overrides needed):

## Overrides

Override any category to customize the Lantern identity. Unspecified values keep their defaults.

<Tabs groupId="lantern-overrides">
  <TabItem value="colors" label="Colors" default>

```ts title="docusaurus.config.ts"
{
  preset: 'lantern',
  overrides: {
    colors: {
      primary:   { light: '#f59e0b', dark: '#f59e0b' },
      secondary: { light: '#4338ca', dark: '#4338ca' },
      text:      { light: '#3d2f1a', dark: '#f0e2c0' },
      border:    { light: '#e8dcb8', dark: '#4a3d24' },
      warning:   { light: '#f59e0b', dark: '#f59e0b' },
      danger:    { light: '#b91c1c', dark: '#b91c1c' },
    },
  },
}
```

Each color category accepts a `{ light, dark }` pair. The `primary` and `secondary` colors expand to 11-level shade scales (50 through 950) as CSS custom properties.

  </TabItem>
  <TabItem value="fonts" label="Fonts">

```ts title="docusaurus.config.ts"
{
  preset: 'lantern',
  overrides: {
    fonts: {
      display: 'Fraunces',
      body: 'Manrope',
      code: 'IBM Plex Mono',
    },
  },
}
```

Font families are set as CSS custom properties. Google Fonts are loaded automatically.

</Tabs>

## Navbar and Footer

Lantern uses the **Bridge** navbar and **Commons** footer.

| Slot   | Variant   | Description                                                                        |
|--------|-----------|------------------------------------------------------------------------------------|
| Navbar | `bridge`  | Horizontal navigation bar carrying Lantern's amber-underline-glow eaves signature. |
| Footer | `commons` | Multi-column footer with sections, social links, and copyright.                    |

Override the variant in the preset options:

```ts title="docusaurus.config.ts"
{
  preset: 'lantern',
  overrides: {
    navbar: 'canopy',
    footer: 'embassy',
  },
}
```

Available navbar variants:

- `bridge`
- `canopy`
- `compass`
- `monolith`

Available footer variants:

- `commons`
- `embassy`
- `launchpad`
- `ledger`
