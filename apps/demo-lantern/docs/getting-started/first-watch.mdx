# Your First Watch

A first watch is the smallest possible perimeter: one zone, one device, one quiet evening.

## Step 1: Discover the Hub

```bash
lantern hub discover
```

Lantern locates the HomeKit Home Hub on the network and pairs without a fuss.

## Step 2: Name a Zone

```bash
lantern zone create perimeter --device front-door
```

A zone with one device is still a zone. Lantern treats it the same way it treats the whole house.

## Step 3: Arm It

```bash
lantern arm perimeter
```

The front door is watched. Lantern says nothing until something happens, and even then, gently.

## What Just Happened

You discovered a hub, named a zone, and armed it. The porch is now part of the watch.
