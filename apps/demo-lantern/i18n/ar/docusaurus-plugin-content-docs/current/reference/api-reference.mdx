# مرجع الواجهة البرمجية

واجهة Lantern Spoke البرمجية صغيرة بقصد. أغلب البيوت لا تحتاجها أبداً.

## عنوان URL الأساسي

```
https://lantern.local/api
```

## نقاط النهاية

| الطريقة | المسار                    | الوصف                                |
|---------|---------------------------|--------------------------------------|
| `GET`   | `/api/zones`              | تسرد كل منطقة يراقبها Lantern.       |
| `POST`  | `/api/zones/:name/arm`    | تفعِّل منطقة بالاسم.                 |
| `POST`  | `/api/zones/:name/disarm` | تلغي تفعيل منطقة بالاسم.             |
| `GET`   | `/api/events`             | تعيد تشغيل تدفّق الأحداث.            |
| `GET`   | `/api/history`            | تقرأ سجلّ المستشعرات.                |
| `GET`   | `/api/mode`               | تجلب وضع البيت الحالي.               |
| `POST`  | `/api/mode`               | تضبط وضع السفر، أو النوم، أو العادي. |

## مثال

```bash
curl https://lantern.local/api/zones \
  -H "Authorization: Bearer $LANTERN_TOKEN"
```

يسرد الردّ كل منطقة، وأجهزتها، وما إذا كانت مفعّلة حالياً. لا شيء أكثر، ولا شيء أعلى.
