# التدقيق والتحقيق الجنائي

Spyglass هو محرّك التدقيق والتحقيق الجنائي في Sentinel. يُسجّل كلّ قرار وصول، وكلّ تقييم سياسة، وكلّ حدث عبور منطقة، وكلّ اتصال نفق Filament، تسجيلاً غير قابل للتعديل لمدّة سبع سنوات. الغرض ليس مجرّد الاحتفاظ، بل إعادة البناء. يمكن إعادة تشغيل أيّ جلسة، أو أيّ حادثة، أو أيّ نمط وصول، خطوة بخطوة، من بيانات السجلّ وحدها.

## ما يُسجّله Spyglass

يلتقط Spyglass كلّ حدث في حلقة الثقة في Sentinel:

| نوع الحدث        | المصدر     | البيانات الملتقطة                                                  |
|------------------|------------|--------------------------------------------------------------------|
| تقييم السياسة    | Watchtower | المستخدم، الجهاز، المورد، الشروط، الدرجة، النتيجة، الطابع الزمني   |
| قرار الوصول      | Drawbridge | منح/تضييق/إلغاء، مُعرّف السياسة، مُعرّف النفق، المدّة              |
| عبور منطقة       | Rampart    | منطقة المصدر، منطقة الهدف، قاعدة العبور، نتيجة تقييم السياسة       |
| فحص الوضع الأمني | Garrison   | مُعرّف الجهاز، جميع فئات الفحص، الدرجات، نجاح/إخفاق لكلّ فئة       |
| حدث النفق        | Filament   | مُعرّف النفق، نقاط النهاية، المدّة، البايتات المنقولة، سبب الإغلاق |
| الوصول الطارئ    | المسؤول    | المستخدم، التبرير، النطاق، المدّة، المُراجِع                       |

جميع الأحداث مُوقّعة تشفيرياً ومُخزّنة في سجلّات للإضافة فقط. ولا يمكن تعديل السجلّات أو حذفها، حتى من قِبَل المسؤولين.

:::info غير قابل للتعديل بحكم التصميم
تُكتب سجلّات Spyglass إلى مخزون للإضافة فقط بسلسلة تشفيرية. يتضمّن كلّ سجلّ تجزئة (هاش) السجلّ السابق، مُكوّناً سلسلة كاشفة للعبث. وأيّ تعديل لسجلّ تاريخي يكسر السلسلة ويُحرّك تنبيه سلامة.
:::

## إعادة بناء الجلسات

يستطيع Spyglass إعادة بناء جلسة وصول كاملة من أحداثها المُكوِّنة. هذه أداة التحقيق الجنائي الأساسيّة في التحقيق في الحوادث:

```bash title="إعادة بناء جلسة"
sentinel spyglass reconstruct \
  --user j.chen \
  --device dev_a3b7c9d1 \
  --period "2025-06-15T09:00:00..2025-06-15T17:00:00"
```

## استعلامات التدقيق

يدعم Spyglass استعلامات منظّمة للبحث عبر جميع أنواع الأحداث:

```bash title="استعلام بالمورد والنطاق الزمني"
sentinel spyglass query \
  --resource api-cluster-east \
  --event-type access-decision \
  --result revoke \
  --period "2025-06-01..2025-06-30"
```


## تصدير الامتثال

يُولّد Spyglass حزم أدلّة امتثال لأُطر العمل القياسية:

```bash title="تصدير أدلّة الامتثال"
sentinel spyglass export --framework soc2 --period 2025-Q2
sentinel spyglass export --framework hipaa --period 2025-Q2
sentinel spyglass export --framework iso27001 --period 2025-Q2
```

يُنتج كلّ تصدير حزمة أدلّة منظّمة تحتوي على ملخّصات تقييم السياسة، وإحصاءات قرارات الوصول، وتوزيعات فحوصات الوضع الأمني، وسجلّات عبور المناطق، مُنسّقة لإطار الامتثال المحدّد.
