# Installation

Lantern installs from the Loom Registry or as a Vial container. Pick the path your hub already trusts.

## Prerequisites

- A HomeKit Home Hub on the network
- Spark 3.0 or later (for CLI installation)
- A quiet ten minutes — most installs finish before the kettle does

## Install via Spark

```bash title="Install Lantern"
spark install lantern
```

Verify the install:

```bash
lantern --version
```

## Install via Vial

```bash title="Pull the Lantern image"
vial pull lantern/cli:latest
```

Run inside the home network's container plane. Lantern keeps to its own room.
