# Unlisted Sample

Use this format for any incident that triggers a page. Keep it short. Two sentences on what happened. Two sentences on what the user impact was. One sentence on the immediate fix.

Then the section that actually matters: what failed in our detection pipeline. Did we get a page before the user noticed? If yes, was it a useful page or a noisy one? If no, what telemetry was missing that would have caught it?

Do not bother with elaborate timelines unless there is a regulatory reason. We are trying to learn something, not produce a report for an auditor.
