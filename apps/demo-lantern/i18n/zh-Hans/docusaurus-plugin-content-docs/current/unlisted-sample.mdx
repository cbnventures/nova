# 未列出示例

这一页并不在已发布的文档之内，但当读者读完安装指南、来问"从哪里继续读"时，我们递给他们的就是这一份。

清单上的第一本永远是谷崎润一郎的《阴翳礼赞》——它与家庭自动化毫无关系，却重新定义了如何把照明视为氛围而非照度。

第二本是巴什拉的《空间的诗学》——它的每一章都在谈门槛，而智能锁正好就是一道门槛。第三本则是当期《Make》杂志中关于家用电子的那一期。

这份清单带有偏见。那正是它的意义。
