# 草稿样例

关于 v3 intake 表单改版的笔记。当前表单询问管辖范围、立卷方、文件类别与参考案卷编号——但我们一直在与提交者反复沟通，因为他们并不清楚自己的记录属于哪一种文件类别，而当前下拉框有三十一项可选。

往下推进的路径，大概要么是把类别分类法收敛为七个顶层归并，再以渐进式披露展开；要么用一个自由文本字段替代下拉框，并在 intake 复核时统一对齐。

无论怎样，这种反复沟通平均每件提交要耗费两个工作日。我们会先调出上季度的指标，再做决定。
