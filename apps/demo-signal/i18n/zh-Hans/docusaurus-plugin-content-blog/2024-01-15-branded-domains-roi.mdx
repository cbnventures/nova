我们用为期六周的 A/B 测试、跨 120 万次链接曝光，去衡量品牌短域名对点击率的影响。通用链接使用 `go.link/` 前缀；品牌链接使用客户自己的域名。结果之间的差距不容忽视。

在我们测试过的每一个渠道上，品牌域名平均带来 34% 更高的点击率。邮件场景下提升最大（41%），付费社交最小（18%）——这其实说得通：邮件收件人在点击之前有更多时间评估一条链接。

## 测试设置

我们与三家中端市场 SaaS 公司合作，把它们的链接流量平均拆分到通用 Beacon 链接与品牌 Beacon 链接两组。其它变量保持完全一致：相同的文案、相同的位置、相同的受众分群。

| 渠道   | 通用链接 CTR | 品牌链接 CTR | 提升幅度 |
|------|----------|----------|------|
| 邮件   | 3.2%     | 4.5%     | 41%  |
| 社交   | 1.1%     | 1.4%     | 27%  |
| 付费社交 | 0.8%     | 0.9%     | 18%  |
| 短信   | 4.6%     | 6.1%     | 33%  |
| 应用内  | 2.3%     | 3.2%     | 39%  |

短信的提升排在第二。在一块上下文非常有限的小屏幕上，可识别的域名几乎是为数不多的信任信号之一。扫一眼短信的用户会在不到一秒内决定一条链接是否值得点开。

## 信任是一种点击放大器

机制并不复杂。读起来像 `go.acme.com/pricing` 的链接，比 `go.link/x8f3m` 自带更多隐性信任。用户认得这个域名，并把它和一个已经在交互的品牌联系起来。点击的心理成本就此下降。

这不是一项品牌建设练习，而是一根转化杠杆。把 34% 的 CTR 提升乘上每月一百万次点击，就是 34 万次本可能因为一瞬间犹豫而流失的有效会话。

## 在 Beacon 中配置品牌域名

Beacon 支持自定义品牌域名，并自动签发 SSL 证书：

```json title="品牌域名配置"
{
  "domain": "go.acme.com",
  "ssl": "auto",
  "fallback": "https://acme.com",
  "attribution": {
    "defaultChannel": "branded",
    "tracePrefix": "acm"
  }
}
```

DNS 配置只需要一条指向 Beacon 边缘网络的 CNAME 记录。SSL 证书会自动签发与轮换。绝大多数团队在更新 DNS 之后十五分钟内就可以上线。

## 复利效应

品牌域名还会改善链接的转发行为。当用户把 `go.acme.com` 链接转给同事时，对方看到的是一个可识别的域名——而不是一段像垃圾邮件的随机短链。这会减少转发链接上的暗流量惩罚，并保留更多归因链。

ROI 的论证非常简单：配置品牌域名几乎不需要成本，却能改善点击之后的每一个指标。
