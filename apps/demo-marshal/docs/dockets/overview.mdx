# Dockets

A docket is the public record of every action Marshal has taken on a repository. Every closed thread, every stamped approval, every scheduled rollover writes a line here.

## Docket entries

- **FILED** — a workflow was queued for execution
- **STAMPED** — Marshal acted, and the result is now part of the record
- **RECEIVED** — incoming work, awaiting tier classification
- **ARCHIVED** — a thread was sealed; the case is closed

The docket is append-only. Marshal does not retract. Marshal amends only by filing a new entry.
