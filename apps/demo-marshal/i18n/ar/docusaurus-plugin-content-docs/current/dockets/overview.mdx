# اللوائح

اللائحة هي السجل العام لكلّ إجراء قام به Marshal على مستودعٍ ما. كلّ مسار مُغلَق، وكلّ موافقة مختومة، وكلّ تجديد مجدوَل يُسجِّل سطرًا هنا.

## بنود اللائحة

- **FILED** — قُيِّد سير عمل في طابور التنفيذ
- **STAMPED** — تصرّف Marshal، والنتيجة أصبحت الآن جزءًا من السجل
- **RECEIVED** — عمل وارد بانتظار تصنيف فئته
- **ARCHIVED** — مسار مختوم، والقضيّة مُغلَقة

اللائحة تُضاف فحسب. لا يَسحَب Marshal شيئًا. لا يُعدِّل Marshal إلّا بإيداع بندٍ جديد.
