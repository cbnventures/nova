# 草稿示例

下一批接头的对比说明。304 不锈钢能达到我们想要的成本，在室内饮用水管线上完全够用，但客户的标书要求 316——因为管线要穿过一处沿海设施，存在氯化物暴露。

价差大约是棒料三十个百分点、条料再加十五个百分点。粗算下来，每套部件升级用料要多吃约四千美元的成本，这一批撑得住，但如果常态化，对利润是一种侵蚀。

我们得在动料之前向采购部确认：海洋等级的规格究竟是合同硬要求，还是仅仅停留在愿望层面。
