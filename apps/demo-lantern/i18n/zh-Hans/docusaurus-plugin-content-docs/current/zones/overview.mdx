# 分区

分区是一个被命名的房间、过道、外部区域或电器组合，由 Lantern 一并看守。请按设备所在位置归类，而非按它们是什么。

## 默认分区

- **Perimeter（外周）** — 外部门、窗、门禁
- **Interior（室内）** — 过道、楼梯间、公共房间
- **Watch（守望）** — 入睡区域，夜幕之后
- **Quiet（安静）** — 卧室与卫生间——仅在旅行模式下布防

## 为分区布防

每个分区都可独立布防。撤防时温柔地按名呼叫；用一句口令也能覆盖。Lantern 从不提高音量。
