# 传感器历史

传感器历史是 Lantern 所见过的每一次动作、接触与分区变更的安静账册。请在正午阅读它，也可以在漫长假期之后翻一翻。这盏灯始终把页面摊开。

## 历史记录了什么

- **动作轨迹** — 方向、置信度、所处时段
- **接触停留** — 后门被推开后维持了多久
- **布防与撤防节奏** — 何时谁布防、何时谁撤防
- **静默时段** — 中间那些空白

## 阅读账册

```bash
lantern history --zone perimeter --since 7d
```

输出只是一列时间与一些温和的动词。任何细节都不会被概括掉。
