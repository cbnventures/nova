Run all enabled <Terminology title="transform script" to="/docs/quickstart/terminology#nova-concepts" color={true}>recipes</Terminology> in sequence.

## Summary

The `run-recipes` command discovers and executes all registered <Terminology title="transform script" to="/docs/quickstart/terminology#nova-concepts" color={true}>recipes</Terminology> in sequence. This is the single command that replaces chaining individual <Terminology title="transform script" to="/docs/quickstart/terminology#nova-concepts" color={true}>recipe</Terminology> commands.

## Why Use This Command?

- **One command to rule them all** — Instead of chaining `nova recipe package-json sync-identity && nova recipe package-json sync-ownership && ...`, run a single `nova utility run-recipes`.
- **Consistent ordering** — Recipes execute in their registered order, ensuring dependencies between recipes are respected.

## Requirements

- **Node.js runtime** — Use any Node.js <Terminology title="long term support" to="/docs/quickstart/terminology#fundamentals" color={true}>LTS</Terminology> release with either the installed `nova` <Terminology title="command line interface" to="/docs/quickstart/terminology#fundamentals" color={true}>CLI</Terminology> or `npx`.
- <Terminology title="Nova config" to="/docs/quickstart/terminology#nova-concepts" color={true}>`nova.config.json`</Terminology> — A valid <Terminology title="Nova config" to="/docs/quickstart/terminology#nova-concepts" color={true}>`nova.config.json`</Terminology> file must exist at the project root with configured <Terminology title="package" to="/docs/quickstart/terminology#nova-concepts" color={true}>workspaces</Terminology>.

## Usage

<Tabs groupId="install-state">
  <TabItem value="nova" label="nova (installed)" default>

    ```bash
    # Original
    nova utility run-recipes [options]

    # Shorthand
    nova util run-rcp [options]
    ```

  </TabItem>
  <TabItem value="npx" label="npx (no install)">

    ```bash
    npx --yes @cbnventures/nova@latest utility run-recipes [options]

    # Shorthand
    npx --yes @cbnventures/nova@latest util run-rcp [options]
    ```

  </TabItem>

## Options

| Flag                 | Description                                                            |
|----------------------|------------------------------------------------------------------------|
| `-d, --dry-run`      | Preview changes without modifying files.                               |
| `-r, --replace-file` | Overwrite the original file instead of creating a `.nova-backup` copy. |
