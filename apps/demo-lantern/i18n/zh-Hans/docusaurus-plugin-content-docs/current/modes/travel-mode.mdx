# 旅行模式

旅行模式会收紧规则。Quiet 分区变成 Watch 分区。睡前窗口关上。门廊的灯会学着在这几天里多一点疑心。

## 进入旅行模式

```bash
lantern mode travel --until 2026-05-01
```

结束日期是必填的。Lantern 不会比你要求的更长时间地保持警觉。

## 它会改变什么

- **Quiet 分区** 持续布防
- **置信带** 收窄——细小的动静也会计入
- **接力** 必须凭明确的暗语
- **通知** 仅路由到一处可信集线器

## 回到家中

旅行模式会在你设定的日期结束，或在你用一台已识别的设备布防 `home` 时提前结束。Lantern 会自动柔下来。
