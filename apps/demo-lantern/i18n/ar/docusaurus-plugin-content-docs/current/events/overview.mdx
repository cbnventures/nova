# الأحداث

كل حركة، وكل اتصال، وكل تفعيل أو إلغاء يمرّ عبر تدفّق أحداث Lantern. أعد التشغيل، رشِّح، سلِّم إلى موزّع. لا شيء يُهمَل، ولا شيء يُصرَخ به.

## أنواع الأحداث

- **الحركة** ـ ذات اتجاه، مع نطاق ثقة اختياري
- **الاتصال** ـ باب أو نافذة فُتحت أو أُغلقت أو ظلّت محجوزة
- **التفعيل والإلغاء** ـ حسب المنطقة، أو المستخدم، أو الجدول
- **سجل المستشعرات** ـ سجلّ هادئ يمكنك قراءته عند الظهيرة

## الخطوات التالية

- [سجل المستشعرات](/docs/events/sensor-history/) ـ السجلّ الهادئ لكل حدث.
- [التكاملات](/docs/zones/integrations/) ـ وجِّه الأحداث إلى موزّع أو إلى نقطة نهاية Spoke.
- [مرجع الواجهة البرمجية](/docs/reference/api-reference/) ـ أعد تشغيل الأحداث برمجياً.
