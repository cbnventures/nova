# 接力

接力让一位可信赖的人，无需你的手机在手便能为分区撤防。Lantern 接受这次接管，记录下持令人，并让灯继续亮着。

## 接力方式

- **按名** — 家庭花名册识得这位的声音
- **按暗语** — 一句事先约好的轻声口令
- **按 Home Hub 在场** — 集线器本就信任这个人
- **按时间表** — 一段会自行打开、又自行关闭的信任窗口

## 接力到来时

```bash
lantern handoff list
```

每一次撤防，都会成为事件流中的一行。这盏灯，记得谁回了家。
