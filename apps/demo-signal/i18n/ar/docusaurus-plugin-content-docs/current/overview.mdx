# Signal

Signal هو محرك إسناد لاقتصاد الانتباه. يتتبع كيف تنتقل النية، من الظهور الأول إلى الإجراء النهائي. كل رابط مستشعر. كل إعادة توجيه نقطة بيانات. لا يخبرك Signal فقط بما تم النقر عليه، بل بسبب ذلك، وبما كاد يحدث بدلاً منه.

> "أقصر مسافة بين النقرة والرؤية هي رابط يعرف أين كان."

## ماذا يفعل Signal

تعدّ معظم أدوات التحليلات الأحداث بعد وقوعها. أما Signal فيستشعر المسار نفسه. كل رابط يُصدَر عبر Signal يحمل بيانات وصفية مضمنة للإسناد تبقى صامدة عبر اختصار عناوين URL وتطبيقات المراسلة والتصفح الخاص والتنقل بين الأجهزة. عند حدوث تحويل، يعيد Signal بناء الرحلة الكاملة، كل نقطة تماس، كل تردد، وكل قناة أسهمت.

| الأداة        | الغرض                                                                                   |
|---------------|-----------------------------------------------------------------------------------------|
| **Beacon**    | محرك إنشاء الروابط، روابط قصيرة موسومة بالعلامة التجارية مع بيانات وصفية مضمنة للإسناد. |
| **Prism**     | نموذج إسناد متعدد اللمس، يتتبع مسارات التحويل عبر القنوات.                              |
| **Pulse**     | تدفق نقرات لحظي، بثّ مباشر للتفاعلات والموقع الجغرافي والجهاز والمُحيل.                 |
| **Aperture**  | لقطات الحملات، التقاطات للأداء في لحظات محددة مع طبقات للمقارنة.                        |
| **Resonance** | كاشف تداخل الجمهور، يكتشف أين تتسرّب الجماهير بين بعضها.                                |
| **Flare**     | مولّد رموز QR والروابط العميقة، جسر من العالم المادي إلى الرقمي مع إسناد كامل.          |

:::info
يعمل Signal على مستوى الرابط نفسه. تثبّت منصات الإسناد الأخرى التتبع على الصفحات. أما Signal فيُضمّنه داخل الروابط ذاتها، فتوجد بيانات الإسناد قبل أن تُحمَّل الصفحة أصلاً.
:::

## كيف يعمل

رابط Beacon ليس مجرد إعادة توجيه. يُشفِّر الحملة والقناة والصيغة ومعرّف تتبع فريد. عند النقر على الرابط، يلتقط Pulse التفاعل لحظياً. وعندما يُحوّل المستخدم في النهاية، يسير Prism في مسار الإسناد الكامل عكسياً ويوزّع الفضل على كل نقطة تماس.

```
Beacon ينشئ الرابط ← المستخدم ينقر ← Pulse يلتقط الحدث
  ← المستخدم ينقر مجدداً من قناة مختلفة ← Pulse يلتقط
    ← المستخدم يُحوّل ← Prism يعيد بناء المسار الكامل
      ← Aperture يلتقط حالة الحملة في لحظة التحويل
```

## متى تستخدم Signal

- **إسناد الحملات** — تتبّع القنوات والروابط ونقاط التماس التي تقود التحويلات.
- **استخبارات الجمهور** — اكتشاف التداخل بين الحملات والشرائح عبر Resonance.
- **الرصد اللحظي** — مشاهدة تدفقات النقرات تصل مباشرة عبر Pulse.
- **من المادي إلى الرقمي** — ربط المطبوعات والإعلانات الخارجية والفعاليات بالإسناد الرقمي عبر Flare.
- **استرداد الحركة المظلمة** — استعادة الإسناد للنقرات التي تُجرَّد من بيانات المُحيل وUTM.
