# Events

Every motion, contact, and arm/disarm passes through Lantern's event stream. Replay, filter, hand off to a hub. Nothing is dropped, nothing is shouted.

## Event types

- **Motion** — directional, with an optional confidence band
- **Contact** — door or window opened, closed, held
- **Arm / disarm** — by zone, by user, by schedule
- **Sensor history** — a quiet ledger you can read at noon
